# Project Summary

## Recommended repository name

**constraint-satisfaction-futoshiki-solver**

This name is hiring-manager-friendly because it clearly communicates constraint satisfaction, algorithmic search, Futoshiki solving, and a reusable solver implementation.

## Alternative names

- `python-csp-futoshiki-solver`
- `ai-constraint-propagation-solver`
- `backtracking-gac-futoshiki`
- `csp-search-algorithms-python`

## GitHub short description

Python CSP solver implementing backtracking, Forward Checking, GAC, and MRV for Futoshiki and N-Queens.

## Resume bullets

- Implemented a Python constraint satisfaction solver with backtracking, Forward Checking, Generalized Arc Consistency, and MRV variable ordering for Futoshiki and N-Queens.
- Modeled Futoshiki puzzles using both binary row/column constraints and n-ary all-different constraints, enabling comparison of propagation strength and search efficiency.
- Built reusable CSP components for variables, domains, constraints, satisfying tuples, pruning/restoration, and automated solver testing.
